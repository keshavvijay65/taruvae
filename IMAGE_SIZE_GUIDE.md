# Image Size Guide for Taruvae Website

## Product Images

**Recommended Size: 400x400 pixels**

All product images should be:
- **Width:** 400px
- **Height:** 400px
- **Format:** JPG, PNG, or WebP
- **Aspect Ratio:** 1:1 (Square)
- **File Size:** Optimize to keep under 200KB for faster loading

## Why 400x400px?

- Perfect for product cards across all pages
- Maintains consistency across home page, products page, and admin panel
- Responsive design automatically scales down on mobile devices
- Optimal balance between quality and file size

## Image Placement

Place all product images in:
```
/public/images/products/
```

## Image Optimization Tips

1. Use tools like TinyPNG or ImageOptim to compress images
2. Use WebP format for better compression (if supported)
3. Ensure images have transparent or white backgrounds for better display
4. Keep product centered in the image frame
