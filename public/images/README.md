# Images Directory Structure

This directory contains all static images used in the Taruvae website.

## Folder Structure

- **products/** - Product images (oils, ghee, seeds, powders, etc.)
- **collections/** - Collection/category images
- **icons/** - Icon files and logos
- **hero/** - Hero section images and banners

## Usage

Images in the `public` folder can be referenced directly in your code:

```tsx
// Example usage
<img src="/images/products/rice-bran-oil.jpg" alt="Rice Bran Oil" />
```

## Image Guidelines

- Use optimized images (WebP, JPEG, PNG)
- Recommended sizes:
  - Product images: 800x800px or 1000x1000px
  - Collection images: 600x600px
  - Hero images: 1920x1080px or larger
- Keep file sizes under 500KB when possible
